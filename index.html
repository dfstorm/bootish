Installation

Simply add this to a form:

<!-- bootish -->
<div id="bootish-container">Loading...</div>
<script>
var bootishSetup = {
	sFormAction : '/post.php' // set the form action.
}
</script>
<script src="//bootish.genois.tk/l.js"></script>
<!-- bootish / -->

To acheive a minimum of security, add a dummy form action and set the good one to "sFormAction" parametter

Options

<!-- bootish Options -->
<script>
// Bootish options are optional.
// Must be placed before the loading script.
	var bootishSetup = {
		sFormAction : '',
		iWidth : 250,

		fncLoaded : function() {},

</script>
