Installation

Simply add this to a form:

<!-- bootish -->
<div id="bootish-container">Loading...</div>
<script src="//bootish.genois.tk/l.js"></script>
<!-- bootish / -->

Documentations

Right now the code isn't open for safety reason. But it will.

Initialisation

  • Remove the form Action
  • Add the attribute "disabled" to every "submit" input.
  • Create an iFrame to https://bootish.genois.tk/ that display the button control.

Waiting

  • Keep a link with the server with ajax requests

User action recived

  • After validation, the key at "bootish.genois.tk" is signed. Then the native JS from l.js re-activate the form by setting the from action back and remove disabled attribute it added.
